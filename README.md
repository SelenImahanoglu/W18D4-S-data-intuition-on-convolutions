# Konvolüsyon Sezgileri

Şimdi Bilgisayarlı Görü konusunda daha sağlam bir anlayış geliştirmek için harika bir zaman.

Evrişimli Sinir Ağı (Convolutional Neural Network) görüntüleri nasıl analiz eder?

Bu `Konvolüsyon Sezgileri` meydan okumada şunları yapacaksınız:
- görüntülere çekirdekler (kernel) uygulama
- bir görüntünün ConvNet tarafından evrişimli katmanları (`layers.Conv2D`) aracılığıyla nasıl algılandığını keşfetme.
